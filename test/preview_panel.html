Lancez un script Python avec plt.show() pour afficher vos figures ici.

  • Les nouveaux graphes apparaissent dans l historique.
  • Cochez plusieurs graphes pour les comparer cote a cote ou en superposition.
  • Utilisez Enregistrer ou Copier directement depuis chaque carte.
  • Importez un fichier .csv/.dat via le bouton Importer CSV (barre d'outils = nouveau graphe ; menu d'un graphe = superposer). Le glisser-deposer marche aussi, mais maintenez Shift (sinon VS Code ouvre le fichier dans l'editeur).
  • Une image .png/.svg exportee par Chaz Plots est auto-portee : reconstruisez le code matplotlib qui reproduit la courbe via le bouton Image → code (ou en deposant l'image avec Shift).
